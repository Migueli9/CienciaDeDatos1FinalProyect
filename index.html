Webscraping

El web scraping es una técnica utilizada para extraer información de sitios web de forma automatizada. En lugar de copiar manualmente datos de una página web, el web scraping permite a los desarrolladores usar scripts o programas para obtener grandes cantidades de datos de manera rápida y estructurada.

¿Cómo funciona?

El proceso básico del web scraping implica:

  1. Enviar una solicitud a una página web: Se usa un programa (como requests en Python) para hacer una solicitud HTTP al sitio web del que se desea extraer información.
  2. Obtener el contenido HTML: El servidor responde con el código HTML de la página.
  3. Extraer la información deseada: Con bibliotecas como BeautifulSoup (Python), se puede analizar y extraer datos específicos, como texto, imágenes, enlaces, tablas, etc., de la estructura HTML.
  4. Almacenar los datos: Los datos extraídos se pueden guardar en diferentes formatos, como bases de datos, archivos CSV o JSON, dependiendo de la aplicación.

Tecnicas de visualizacion

La visualización de datos es una parte fundamental en la ciencia de datos, ya que permite comunicar de manera clara y eficiente los hallazgos obtenidos de los análisis. Existen diversas técnicas que se utilizan dependiendo del tipo de datos y la información que se quiera destacar. A continuación, te explico algunas de las técnicas más comunes y efectivas:

  1. Gráficos de barras (Bar Charts)

    2. Los gráficos de barras son ideales para comparar diferentes categorías. Cada barra representa una categoría, y su altura o longitud indica el valor que representa.

    • Cuándo usarlo: Comparar cantidades en diferentes grupos.
    • Ejemplo: Comparar ventas por producto, frecuencia de eventos en diferentes grupos.

    Herramientas Comunes:

    • Matplotlib (Python)
    • Excel
    • Power BI

  2. Histogramas

    3. Un histograma es similar a un gráfico de barras, pero se usa para mostrar la distribución de un conjunto de datos continuos. Se agrupan los valores en rangos (bins), y la altura de cada barra indica la frecuencia de datos en ese rango.

    • Cuando usarlo: Mostrar la distribución de una variable numérica continua.
    • Ejemplo: Distribución de ingresos en una población, distribución de edades en un grupo.

    Herramientas Comunes:

    • Matplotlib (Python)
    • Excel

  3. Graficos de pastel

    4. Los gráficos de pastel muestran las proporciones de un todo. Cada sector representa una parte de un conjunto total.

    • Cuándo usarlo: Mostrar proporciones o porcentajes de un conjunto total.
    • Ejemplo Distribución de votos en una elección, participación de mercado por empresa.

    Herramientas Comunes:

    • Matplotlib (Python)
    • Excel

Estas técnicas te permitirán analizar y comunicar datos de manera efectiva, dependiendo del tipo de análisis que estés realizando. Cada técnica tiene su fortaleza, y es importante seleccionar la que mejor se ajuste al tipo de información que deseas transmitir.

  2. Mediana

    3. La mediana es el valor que divide el conjunto de datos en dos partes iguales, de manera que la mitad de los datos son menores o iguales que la mediana, y la otra mitad son mayores o iguales.

    ¿Como se calcula?

    • Si el número de datos es impar, la mediana es el valor del medio cuando los datos están ordenados.
    • Si el número de datos es par, la mediana es el promedio de los dos valores centrales.

  3. Moda

    4. La moda es el valor o valores que aparecen con mayor frecuencia en un conjunto de datos. Un conjunto de datos puede tener una moda, más de una moda o ninguna moda.

    Ejemplo:

    • Para los datos: 10, 15, 15, 20, 25, la moda es 15 (porque aparece más veces).
    • Si tienes: 10, 10, 15, 15, 20, 25, el conjunto es bimodal (dos modas: 10 y 15).
    • Si los valores son todos diferentes, no hay moda.

Mediana No afectada por outliers, solo toma en cuenta el valor central. Cuando los datos tienen valores extremos o distribuciones asimétricas.
Moda Representa el valor más frecuente, útil para datos categóricos. Cuando deseas conocer la frecuencia de ocurrencia de los valores.
